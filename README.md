# 🚀 GitHub Profile Finder

A simple and interactive web application that allows users to search for any GitHub profile using a username.
The application fetches real-time data from the **GitHub API** and displays useful profile information like avatar, bio, followers, repositories, and more.

---

## ✨ Features

* 🔍 Search any GitHub username
* 👤 Display user profile picture
* 📝 Show bio and profile information
* 👥 Display followers and following count
* 📦 Show number of public repositories
* 🔗 Direct button to visit the GitHub profile

## ⚙️ How It Works

1. Enter a GitHub username in the search box.
2. Click the **Search** button.
3. The application sends a request to the **GitHub API**.
4. The API returns the user's profile data.
5. The profile details are displayed on the screen.
