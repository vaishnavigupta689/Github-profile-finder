# GitHub Profile Finder

A simple web application that allows users to search for any GitHub user's profile using their username. The app fetches data from the GitHub API and displays basic profile details like avatar, bio, followers, and repositories.

---

## 🚀 Features

- Search any GitHub username  
- Displays profile picture  
- Shows bio and basic user information  
- Displays followers, following, and public repositories  
- Direct button to open the GitHub profile  

---

## 🛠 Technologies Used

- HTML  
- CSS  
- JavaScript  
- GitHub REST API  

---

## ⚙️ How It Works

1. Enter a GitHub username in the search box.
2. Click the **Search** button.
3. The app sends a request to the GitHub API.
4. The profile information is displayed on the screen.
